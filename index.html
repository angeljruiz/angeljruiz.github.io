// Recent Project

Dendro

Dendro is an open-source, serverless monitoring framework for small, distributed applications.

Dendro is a serverless, time-series native log centralization pipeline built on AWS using their NoSQL Timestream database. Observability in distributed systems is painful. We aim to mitigate that. Dendro automates the deployment of an entire ETL log pipeline to the cloud in seconds and hooks up your existing distributed system to it with just a few clicks.

BridgeAPI

I collaborated remotely to create BridgeAPI.

BridgeAPI.dev's goal is to provide an intuitive tool to receive, filter, and forward API requests to connect incompatible RESTful APIs and automate workflows. Current options for developers usually require subscribing to bloated services and granting intrusive permissions to connect apps. With BridgeAPI, simply define your desired workflow and direct your vendor webhooks or API calls towards your provided BridgeAPI endpoint.
